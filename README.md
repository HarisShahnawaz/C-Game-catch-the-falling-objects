# 🎮 Catch The Falling Objects — C# Game

A fast-paced desktop game built with C# and Windows Forms 
where you control a paddle to catch falling objects before 
they hit the ground. Features increasing difficulty and score tracking.

## 🛠️ Built With
- C# 
- Windows Forms (.NET)
- Visual Studio

## 🎮 How To Run
1. Clone the repository
2. Open the `.sln` file in Visual Studio
3. Press `F5` to build and run

## ✨ Features
- Increasing difficulty over time
- Score tracking
- Simple keyboard controls
- Addictive gameplay loop
